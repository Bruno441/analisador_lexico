# Linguagem Nexon

## Elementos da Sintaxe

| Operação               | Elemento |
|------------------------|----------|
| Definir função         | 'fnc'    |
| Declarar variável      | 'var:'   |
| Retorno de função      | 'back'   |
| Comentário             | '//'     |
| Condição (if)          | 'si'     |
| Condição (else if)     | 'velsi'  |
| Condição (else)        | 'nisi'   |
| Atribuição             | '->'     |
| Impressão              | 'Imp'    |
| Soma                   | '"+"'    |
| Subtração              | '"-"'    |
| Multiplicação          | '"*"'    |
| Divisão                | '"/"'    |
